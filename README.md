# LUMENFORGE

LUMENFORGE LF-1 是一个运行在浏览器中的影像体素浮雕 3D 打印模拟器。导入图片或选择内置示例后，可以调整成型模式、网格精度与浮雕深度，并观看完整的切片和打印过程。

## 在线体验

https://liko0223.github.io/lumenforge/

## 本地运行

需要 Node.js 22 或更高版本。

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

## 技术栈

- React 19 + TypeScript
- Vite
- React Three Fiber / Three.js
- Tailwind CSS + shadcn/ui
